# mailhog-operator

![mailhog-operator](hack/mailhog-operator-wdn.png "mailhog-operator")

A toy operator that deploys [mailhog](https://github.com/mailhog/MailHog) on [codeready containers](https://github.com/code-ready/crc).

## Install

```bash
make crc-deploy
```
